# JOHN (pre-alpha)

JOHN is a web-based financial management tool.
It is based off the spreadsheet I made for myself in 2024, and is meant to be a simple, easy-to-use, and secure way to manage your money.

## Features
- Track expenses and income in a simple spreadsheet-like interface
- Allocate money towards different expenses or goals using Funds
- and more!
- Save your data over the cloud
- Access your data from anywhere
